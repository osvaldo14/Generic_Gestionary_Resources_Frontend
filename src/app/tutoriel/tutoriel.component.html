Comment utiliser HRM

Qu'est ce qu'un type de ressource ?

  • Un type de ressource permet de réunir plusieurs ressources sous un même nom et sous un même ensemble de caractéristiques.
    Pour créer un type de ressource : Cliquez sur créer -> "Créer un type"


Comment créer une ressource ?

  • /!\ Pour créer une ressource il est nécessaire d'avoir au préalable créer son type/!\.
    Pour créer une ressource : Cliquez sur créer -> "créer une ressource"


Comment faire une réservation ?

  • Cliquez sur "réservation". Ensuite un pop-up vous permettra de choisir un nom, le(s) ressource(s), une date et une heure.


Comment modifier ou supprimer une réservation ?

  • Vous pouvez supprimer une réservation en cliquant sur le bouton "supprimer" dans la liste des réservations


  • Vous pouvez modifier tous les éléments d'une réservation via la liste des réservations, n'oubliez pas de confirmer votre modification en cliquant sur le bouton "modifier"


  • Si vous souhaitez uniquement modifier les horaires des réservations, vous pouvez le faire en drag and drop sur les vues hebdomadaire et journalière


Information sur la version :

Cette application web n'est pas encore dans sa version finale, nous vous remerçions pour votre compréhension.
